# Fonts

Navbar and UI fonts are loaded from the **app** folder via `next/font/local`:

- **Program Narrow:** `app/programnarrow copy/ProgramNarOT-Medium.otf` (navbar)
- **Poppins:** `app/poppins copy/*.ttf` (body / general UI)

See `app/fonts.ts` and `app/layout.tsx`. This `public/fonts/` folder is optional (e.g. for extra assets).
